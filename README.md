# Requirements

## Web

create a .env file

```.env
DATABASE_USER="<user>"
DATABASE_PASSWORD="password"
DATABASE_HOST="localhost"
DATABASE_PORT="<port>"
DATABASE_NAME="<database>"

DATABASE_URL="postgresql://${DATABASE_USER}:${DATABASE_PASSWORD}@${DATABASE_HOST}:${DATABASE_PORT}/${DATABASE_NAME}?schema=public"
```
